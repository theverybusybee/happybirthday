# Happy Birthday Landing — ТЗ и идеи

## Концепция
Персональный лендинг-поздравление с днём рождения для мч.
Стиль: тёплый скрапбук — полароид-фото на скотче, рукописные подписи.
Фишка: фон плавно меняет цвет по мере скролла, подстраиваясь под тему секции.
На фоне плавают иконки его хобби (вместе с сердечками).

---

## Хобби мч
| Хобби           | Иконки на фоне                  | Цвет фона секции                        |
|-----------------|----------------------------------|------------------------------------------|
| Музыка / фортепиано / Бетховен | ♪ ♫ 🎹 🎵            | Тёплый фиолетовый (`#2a1b3d` → `#1a1028`) |
| Химия (работа)  | ⚗ 🧪 ⚛                          | Изумрудно-тёмный (`#0a2e2a` → `#0d1f1d`)  |
| Языки (DE, EN)  | A B Ä Ö ß                        | Кремовый тёплый (основной `#fdf6f0`)      |
| Посткроссинг    | ✉ 📮 ✈                          | Тёплый песочный (`#f5e6d0`)               |
| Блог            | ✍ 📝                            | Кремовый тёплый (основной)                |
| Астрономия / звёзды | ★ ☆ ✦ ·                     | Глубокий ночной (`#0b0e1a` → `#111830`)   |

## Новая структура страницы (план)

Фон плавно перетекает между цветами через CSS-градиенты на стыке секций.

```
┌─────────────────────────────────────────────┐
│  1. HERO (кремовый)                         │
│     "С Днём Рождения!"                      │
│     плавающие: ♥ ★ ♪ (микс всех хобби)      │
├─ градиент кремовый → кремовый ───────────────┤
│  2. ТЁПЛЫЕ СЛОВА (кремовый)                 │
│     3 карточки с пожеланиями                │
├─ градиент кремовый → кремовый ───────────────┤
│  3. ФОТО-ИСТОРИИ (кремовый)                 │
│     5 секций фото + текст, скрапбук-стиль   │
├─ градиент кремовый → фиолетовый ─────────────┤
│  4. МУЗЫКА (тёплый фиолетовый)              │
│     Про его любовь к фортепиано/Бетховену   │
│     плавающие: ♪ ♫ 🎵                        │
│     светлый текст                           │
├─ градиент фиолетовый → изумрудный ───────────┤
│  5. ХИМИЯ (изумрудно-тёмный)                │
│     Про его работу и страсть к химии        │
│     плавающие: ⚗ ⚛ 🧪                       │
│     светлый текст                           │
├─ градиент изумрудный → кремовый ─────────────┤
│  6. ЯЗЫКИ И ПОСТКРОССИНГ (кремовый/песочный)│
│     Про языки, открытки, блог               │
│     плавающие: ✉ Ä ✈ ✍                      │
├─ градиент кремовый → ночной ─────────────────┤
│  7. АСТРОНОМИЯ (глубокий ночной)            │
│     Про звёзды и ночное небо                │
│     Анимированные мерцающие звёзды на фоне  │
│     светлый текст                           │
├─ градиент ночной → кремовый ─────────────────┤
│  8. ФИНАЛ (кремовый)                        │
│     Торт, конфетти, главное поздравление    │
│     ♥                                       │
└─────────────────────────────────────────────┘
```

## Шрифты
- **Playfair Display** — заголовки, цитаты
- **Inter** — основной текст
- **Caveat** — рукописные подписи к фото
